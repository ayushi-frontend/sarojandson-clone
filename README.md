# Saroj & Sons Website Clone

This is a frontend clone of the Saroj & Sons website built using React.js.  
The project is created for learning and practice purposes.

## 🚀 Features

- Responsive design (Mobile + Desktop)
- Modern UI layout
- React component-based structure
- Multi-page navigation using React Router
- Clean and reusable code

## 🛠️ Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3

## 📁 Project Setup

1. Clone the repository:
   git clone https://github.com/your-username/sarojandson-clone.git

2. Install dependencies:
   npm install

3. Run the project:
   npm start

4. Build for production:
   npm run build
